## 1. 分词
```python
import fool
import jieba

text = "一个傻子在苏宁"
fool.cut(text, True) 
# [['一个', '傻子', '在', '苏宁']]

jieba.lcut(text, cut_all=True) # list(jieba.cut(text, cut_all=True))
# ['一个', '傻子', '在', '苏宁']
```
