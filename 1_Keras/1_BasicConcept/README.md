## 1. Tensor
- 定义：张量可以看作是向量、矩阵的自然推广，我们用张量来表示广泛的数据类型
- 阶数：维度/轴/axis

  > 譬如一个矩阵[[1,2],[3,4]]，是一个2阶张量，有两个维度或轴，沿着第0个轴（为了与python的计数方式一致，本文档维度和轴从0算起）你看到的是[1,2]，[3,4]两个向量，沿着第1个轴你看到的是[1,3]，[2,4]两个向量。
